How to add DISQUS comments to Pelican site

Date Wed 14 June 2017 By Alexander Kamyanskiy Tags Disqus / Pelican / Python

It is possible to attach DISQUS comments to blog site.

Setup your DISQUS Account

Before you can use the commenting stuff, you need to register into Disqus. There you find a lot of disqus documentation that you simply not need :-) . Pelican already has done that for you.

  1. Register on DISQUS site

  2. Select ‘I don’t see my platform listed, install manually with Universal Code.

  3. Remember ‘Shortname’=kamyanskiy, it will be used later in site configuration. Fill fields Website name and Website URL.

  4. Check ‘Allow guests to comments’

  5. Finish settings with Add trusted site, my own is kamyanskiy.github.io

  6. Into pelicanconf.py file enable DISQUS, just add these settings:

DISQUS_SITENAME = "kamyanskiy"

Remember, here should be added ‘shortname’ from step 3.
